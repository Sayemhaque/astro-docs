# Typography Documentation

This document outlines the typography styles and guidelines for our project.

## Fonts

### Primary Font

The primary font used for headings and main content is **Roboto**.

```css
font-family: 'Roboto', sans-serif;
```

### Secondary Font

The secondary font used for special emphasis and decorative elements is **Open Sans**.

```css
font-family: 'Open Sans', sans-serif;
```

## Font Sizes

We follow a consistent set of font sizes for various elements throughout the project.

### Headings

- **H1**: 32px
- **H2**: 24px
- **H3**: 20px
- **H4**: 18px
- **H5**: 16px
- **H6**: 14px

### Body Text

- **Body**: 16px
- **Small**: 14px (for footnotes, captions, etc.)

## Font Weights

We use a range of font weights to create visual hierarchy and emphasis.

- **Regular**: 400
- **Medium**: 500
- **Bold**: 700

## Line Heights

Maintaining appropriate line heights ensures readability and visual harmony.

- **Heading Line Height**: 1.2
- **Body Line Height**: 1.5

## Text Styles

### Links

Links are styled in our project to be easily distinguishable from regular text.

```css
color: #007bff;
text-decoration: underline;
```

### Italics

Italic text is used for emphasis or to denote a quote.

```markdown
_Italic text_
```

### Bold

Bold text is used for strong emphasis or headings.

```markdown
**Bold text**
```

## Conclusion

Consistent typography is crucial for a cohesive and visually appealing design. By following these guidelines, we ensure readability and maintain a professional aesthetic across our project.
